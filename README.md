# C++设计模式
一个用C++实现设计模式的实验作业.

共有以下三个实验:
* [创建型模式](./src/lab1/README.md)
* [结构型模式](./src/lab2/README.md)
* [行为型模式](./src/lab3/README.md)

各个文件目录说明如下:
* `docs` ：所有的类图文件存放位置(使用 [plantuml](https://plantuml.com/zh/) 语法进行绘制).
* `src` ：C++源代码实现一共有三个文件夹对应三个实验.
* `tests` ：测试文件目录.
* `.clang-format` ：本项目的代码格式化风格.
